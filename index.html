3

כרטוס

💰
עמלה — הקלדה ידנית חובה: כאשר אין מחיר הסכם והכרטוס הוא במחיר מערכת (מפורסם), חובה להקליד את העמלה ידנית מיד אחרי התמחור — גם כשהיא 0% וגם במעמד שינוי.
🔑
אותיות היוזר (YY) — לא "אותיות הסוכן": אלו האותיות המופיעות בחתימה ב-RH. לא בטוחים? בדקו מול מחלקת ההדרכה מה אותיות היוזר שלכם. שימו לב: צורת תשלום FPCASH תתאפשר אך ורק אם אותיות היוזר תואמות לאותיות LY.
⚠️
Tour Code נכנס אוטומטית בתמחור. בדקו: אותיות ה-Sale&Fly מול אותיות היוזר —
✓ תואמות — לא נוגעים. מכרטסים עם CASH והנקודות נמשכות לפי היוזר שסגר את ההזמנה.
✗ לא תואמות — מיד אחרי התמחור: RTF ← XE לשורת ה-FT האוטומטית ← FT*COMxxxxx//YY ← סגירה. לבצע QA — אחריות הסוכן המבצע.
לא ביטלתם את השורה האוטומטית? יהיו שני טור-קודים והמערכת תיקח את הראשון.
⚠️
הסכמים חדשים של LY — וורלדספאן בלבד: תמחור עם קוד החברה מזין אוטומטית TKT DSG לדאטא, את קוד החברה ל-Endorsement ואת ה-Tour Code.
כרטוס — חובה בפורמט המלא: EZ#CLY#K0#ITCOMxxxxx//YY — כרטוס חלקי ידרוס את ה-Tour Code ונקבל ADM.
✓ חריג: אותיות היוזר תואמות לאותיות אל על והוא שפתח את ההזמנה — ניתן לכרטס עם EZ#CLY#K0 בלבד, כל עוד ה-Tour Code נראה בדאטא ולא נדרס.
לבצע QA — אחריות הסוכן המבצע.
